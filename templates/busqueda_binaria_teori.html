Teoría de la Búsqueda Binaria

La búsqueda binaria es un algoritmo de búsqueda eficiente utilizado para encontrar la posición de un elemento en una lista ordenada. La búsqueda binaria funciona dividiendo repetidamente la mitad de la lista en la que se encuentra el elemento objetivo hasta que se encuentra el elemento o se determina que no está presente.

Proceso de la Búsqueda Binaria

  • Comienza con la lista completa, estableciendo los límites inferior y superior.
  • Encuentra el elemento medio de la lista y lo compara con el elemento objetivo.
  • Si el elemento objetivo es igual al elemento medio, la búsqueda se detiene y se devuelve la posición del elemento.
  • Si el elemento objetivo es menor que el elemento medio, repite el proceso en la sublista izquierda.
  • Si el elemento objetivo es mayor que el elemento medio, repite el proceso en la sublista derecha.
  • Este proceso se repite hasta que se encuentra el elemento objetivo o se determina que no está presente.

Características de la Búsqueda Binaria

  • Mucho más eficiente que la búsqueda lineal para listas grandes debido a su complejidad de tiempo O(log n).
  • Requiere que la lista esté ordenada previamente.
  • Implementación relativamente sencilla y fácil de entender.

Complejidad de la Búsqueda Binaria

  • Peor caso: O(log n), cuando el elemento se encuentra en una de las mitades más pequeñas de la lista.
  • Mejor caso: O(1), cuando el elemento está en la posición media inicial.
  • Promedio: O(log n), en el caso general.

Algoritmo de la Búsqueda Binaria

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
