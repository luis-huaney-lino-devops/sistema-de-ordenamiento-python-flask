Teoría del Radix Sort

Radix Sort es un algoritmo de ordenación no comparativo que ordena los números procesando los dígitos individualmente. Comienza ordenando los dígitos menos significativos y avanza hasta los dígitos más significativos. Puede usar Counting Sort como subrutina para clasificar los dígitos.

Proceso del Radix Sort

  • Encuentra el número máximo en la lista para determinar el número de dígitos.
  • Usa una ordenación estable (como Counting Sort) para ordenar los números según cada dígito, comenzando por el dígito menos significativo.
  • Repite el proceso para cada dígito más significativo hasta que todos los dígitos hayan sido procesados.

Características del Radix Sort

  • Es eficiente para listas grandes de enteros con un número limitado de dígitos.
  • No es un algoritmo de comparación.
  • Requiere memoria adicional para la ordenación de dígitos individuales.
  • Es un algoritmo estable, manteniendo el orden relativo de los elementos con valores iguales.

Complejidad del Radix Sort

  • Peor caso: O(d*(n+k)), donde n es el número de elementos, k es el rango de los valores, y d es el número de dígitos en el número más largo.
  • Mejor caso: O(d*(n+k))
  • Promedio: O(d*(n+k))

Algoritmo del Radix Sort

def counting_sort_for_radix(arr, exp):
   n = len(arr)
   output = [0] * n
   count = [0] * 10
   for i in range(n):
     index = arr[i] // exp
     count[index % 10] += 1
   for i in range(1, 10):
     count[i] += count[i - 1]
   i = n - 1
   while i >= 0:
     index = arr[i] // exp
     output[count[index % 10] - 1] = arr[i]
     count[index % 10] -= 1
     i -= 1
   for i in range(n):
     arr[i] = output[i]

def radix_sort(arr):
   max_val = max(arr)
   exp = 1
   while max_val // exp > 0:
     counting_sort_for_radix(arr, exp)
     exp *= 10
