Teoría de la Búsqueda Lineal

La búsqueda lineal es un algoritmo de búsqueda simple que recorre una lista de elementos de principio a fin para encontrar un elemento específico. Este método no requiere que la lista esté ordenada y puede aplicarse a cualquier tipo de estructura de datos secuencial.

Proceso de la Búsqueda Lineal

  • Comienza en el primer elemento de la lista.
  • Compara el elemento objetivo con el elemento actual de la lista.
  • Si el elemento objetivo es igual al elemento actual, la búsqueda se detiene y se devuelve la posición del elemento.
  • Si el elemento objetivo no es igual al elemento actual, la búsqueda continúa con el siguiente elemento de la lista.
  • Este proceso se repite hasta que se encuentra el elemento objetivo o se alcanza el final de la lista.

Características de la Búsqueda Lineal

  • Fácil de entender e implementar.
  • No requiere que la lista esté ordenada.
  • Ineficiente para listas grandes debido a su complejidad de tiempo O(n).
  • Es un algoritmo robusto que puede manejar listas con cualquier tipo de elementos.

Complejidad de la Búsqueda Lineal

  • Peor caso: O(n), cuando el elemento está al final de la lista o no está presente.
  • Mejor caso: O(1), cuando el elemento está en la primera posición de la lista.
  • Promedio: O(n), en el caso general.

Algoritmo de la Búsqueda Lineal

def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target:
            return i
    return -1
