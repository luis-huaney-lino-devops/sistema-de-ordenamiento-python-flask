Teoría del Insert Sort

Insert Sort es un algoritmo de ordenación simple que construye la lista ordenada un elemento a la vez, tomando un elemento de la lista de entrada y encontrando su posición correcta en la lista ya ordenada. Este proceso se repite hasta que todos los elementos estén en la lista ordenada.

Proceso del Insert Sort

  • Empieza con el segundo elemento de la lista (considerando el primer elemento ya ordenado).
  • Compara el elemento actual con los elementos de la lista ordenada.
  • Inserta el elemento actual en la posición correcta dentro de la lista ordenada.
  • Repite el proceso para cada elemento de la lista de entrada.

Características del Insert Sort

  • Fácil de entender e implementar.
  • Requiere memoria adicional mínima.
  • Más eficiente que otros algoritmos simples como Bubble Sort y Selection Sort para listas pequeñas o casi ordenadas.
  • Es un algoritmo estable, lo que significa que mantiene el orden relativo de los elementos con valores iguales.

Complejidad del Insert Sort

  • Peor caso: O(n²)
  • Mejor caso: O(n) cuando la lista ya está ordenada
  • Promedio: O(n²)

Algoritmo del Insert Sort

def insert_sort(arr):
   for i in range(1, len(arr)):
     key = arr[i]
     j = i - 1
     while j >= 0 and key < arr[j]:
       arr[j + 1] = arr[j]
       j -= 1
     arr[j + 1] = key
