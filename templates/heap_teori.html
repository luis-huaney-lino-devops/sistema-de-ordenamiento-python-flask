Teoría del Heap Sort

Heap Sort es un algoritmo de ordenación basado en una estructura de datos llamada heap (montículo), que es una estructura de árbol binario completo. El algoritmo convierte la lista en un heap, y luego extrae el elemento máximo repetidamente para construir la lista ordenada.

Proceso del Heap Sort

  • Construir un max heap a partir de la lista desordenada.
  • Intercambiar el elemento raíz (máximo) con el último elemento de la lista.
  • Reducir el tamaño del heap y aplicar heapify para restaurar las propiedades del heap.
  • Repetir el proceso hasta que toda la lista esté ordenada.

Características del Heap Sort

  • Es un algoritmo eficiente para listas grandes.
  • No es un algoritmo estable, lo que significa que no necesariamente mantiene el orden relativo de los elementos con valores iguales.
  • Requiere memoria adicional mínima ya que se realiza in-place.
  • Utiliza una estructura de datos adicional (heap) para gestionar los elementos.

Complejidad del Heap Sort

  • Peor caso: O(n log n)
  • Mejor caso: O(n log n)
  • Promedio: O(n log n)

Algoritmo del Heap Sort

def heapify(arr, n, i):
   largest = i
   l = 2 * i + 1
   r = 2 * i + 2
   if l < n and arr[i] < arr[l]:
     largest = l
   if r < n and arr[largest] < arr[r]:
     largest = r
   if largest != i:
     arr[i], arr[largest] = arr[largest], arr[i]
     heapify(arr, n, largest)

def heap_sort(arr):
   n = len(arr)
   for i in range(n//2 - 1, -1, -1):
     heapify(arr, n, i)
   for i in range(n-1, 0, -1):
     arr[i], arr[0] = arr[0], arr[i]
     heapify(arr, i, 0)
