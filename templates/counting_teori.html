Teoría del Counting Sort

Counting Sort es un algoritmo de ordenación eficiente para listas de enteros dentro de un rango específico. Funciona contando el número de ocurrencias de cada valor en la lista y luego utilizando esta información para colocar los elementos en su posición correcta en una lista ordenada.

Proceso del Counting Sort

  • Encuentra el valor máximo en la lista para determinar el rango de los valores.
  • Inicializa una lista de conteo con ceros, de tamaño igual al valor máximo más uno.
  • Cuenta la ocurrencia de cada valor en la lista original y actualiza la lista de conteo.
  • Modifica la lista de conteo para que cada posición contenga la suma de las ocurrencias anteriores. Esto ayudará a determinar las posiciones finales de los elementos.
  • Construye la lista ordenada colocando cada elemento en su posición correcta, decreciendo el conteo respectivo en la lista de conteo.

Características del Counting Sort

  • Es eficiente para listas con un rango pequeño de valores.
  • No es un algoritmo de comparación.
  • Requiere memoria adicional para la lista de conteo y la lista ordenada.
  • Es un algoritmo estable, manteniendo el orden relativo de los elementos con valores iguales.

Complejidad del Counting Sort

  • Peor caso: O(n + k), donde n es el número de elementos en la lista y k es el rango de los valores.
  • Mejor caso: O(n + k)
  • Promedio: O(n + k)

Algoritmo del Counting Sort

def counting_sort(arr):
   max_val = max(arr)
   count = [0] * (max_val + 1)
   for num in arr:
     count[num] += 1
   index = 0
   for i in range(max_val + 1):
     while count[i] > 0:
       arr[index] = i
       index += 1
       count[i] -= 1
